¿Qué es ECMAScript?

ECMAScript es el estándar que define cómo debe comportarse un lenguaje como JavaScript. Fue creado para asegurar la compatibilidad entre diferentes implementaciones del lenguaje. Actualizado por el comité TC39, incluye nuevas funciones, mejoras y correcciones regularmente.

Versiones de ECMAScript

VersiónAñoNovedades Clave
ES31999Funciones básicas, manejo de errores
ES52009"use strict", métodos de arrays, getters/setters
ES62015let/const, clases, promesas, arrow functions
ES72016includes(), operador exponencial (**)
ES82017async/await, Object.entries/values
ES92018Rest/Spread, mejoras de async
ES102019flat(), trimStart/End(), Object.fromEntries()
ES112020BigInt, nullish coalescing, matchAll()
ES12+2021–2024Operadores lógicos, mejoras de clase, performance

Diferencias Notorias Entre Versiones

Estas versiones introdujeron diferencias clave:

  • ES5: Mejor estructura de código con modo estricto.
  • ES6: Cambios modernos como `let`, clases, promesas.
  • ES8: `async/await` simplificó el trabajo asíncrono.
  • ES11: Introducción de BigInt y operador de encadenamiento opcional.
